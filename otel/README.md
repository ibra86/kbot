# Open-telemetry observability

Configuration for Kbot that sends:

* logs to [OpenTelemetry Collector] and [Loki] 
* metrics to [OpenTelemetry Collector] and [Prometheus]
* taces to [OpenTelemetry Collector] and [Tempo]

## Prerequisites

- [Docker Compose]

## How to run

```bash
./local_run.sh
```
